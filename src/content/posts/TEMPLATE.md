## Helpful Tips for Formatting

### Images
Store your large images in `public/images/` and small thumbnails in `public/thumbnails/`. 
To embed an image in a post, use this format:
`![Image Alt Text](../images/my-image.png)`

### Internal Links
To link to another blog post, use the `/blog/slug` format. For example:
`[Read my other post](/blog/software-artists)`

### Code Blocks
To write code, use standard markdown fences:
```javascript
function helloWorld() {
  console.log("Hello!");
}
```
